# 011

## 要望

CLI の出力に不要なインデントが残るため修正したい。

現象:
- メニュー選択後の `access_token:` 出力が右にずれる
- メニュー選択後の `対象月を入力してね (yyyy-mm):` が右にずれる

要望:
- これらの出力をすべて左揃えにする
- 無駄なインデントをなくす

## 対象範囲

- メニュー描画/終了直後のカーソル位置制御
- アクション実行前の出力位置の安定化

## 非対象

- メニュー項目・機能仕様の変更
- 認証フロー/API ロジックの変更
