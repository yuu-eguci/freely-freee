# 009

## 要望

選択肢 UI の表示が右側へずれているので、すべて左揃えに修正したい。

現状イメージ:

```text
今回は何をしたい? (↑↓で選択 / Enterで決定 / Ctrl+Cで中断)
                                                         > 指定の月の平日に勤怠を詰め込む
                                                                                           指定月の平日勤怠をリセットする
```

期待する表示:

- 選択中の `>` と選択肢テキストを左端から揃える
- 非選択行も同様に左揃え
- 選択操作 (↑↓ / Enter / Ctrl+C) の挙動は変えない

## 対象範囲

- メニュー描画ロジックの設計と実装
- コンソール表示の整列ルール

## 非対象

- メニュー項目の内容変更
- 認証や API 処理の仕様変更
