# 008

## 要望

新しい機能「指定の月の平日をぜんぶ勤怠空っぽにする」を追加する。

- 目的: テストのため、指定月の登録状況をリセットする
- 処理対象の判定条件は 006 + 007 と同じ
  - `day_pattern == "normal_day"`
  - `use_default_work_pattern == true`
- 上記条件を満たす日付に対して、以下を実行する
  - 勤怠を空にする
  - 勤怠タグを空にする

### 勤怠を空にする API

```http
PUT /hr/api/v1/employees/{employee_id}/work_records/{date}?company_id={company_id}
{
  "company_id": {company_id},
  "break_records": [],
  "work_record_segments": []
}
```

### 勤怠タグを空にする API

```http
PUT /hr/api/v1/employees/{employee_id}/attendance_tags/{date}
{
  "company_id": {company_id},
  "employee_attendance_tags": []
}
```

- 月次ループ・判定・ログ・サマリー
