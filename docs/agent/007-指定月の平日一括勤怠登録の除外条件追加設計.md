# 007

## 要望

006 で設計した「指定の月の平日に勤怠を詰め込む」へ、以下の除外条件を追加する。

- 現在の除外条件:
  - `day_pattern != "normal_day"` は処理対象から外す
- 追加したい除外条件:
  - `use_default_work_pattern == false` も処理対象から外す

## 目的

- 会社や本人の個別勤務パターンが適用されている日 (`use_default_work_pattern == false`) には、自動投入を行わない
- 既存の normal_day 判定と合わせて、意図しない上書きを防ぐ

## 対象範囲

- `app/actions/bulk_attendance.py` の日次処理判定
- 設計ノート 006 の判定条件説明・ログ仕様の更新
- 必要ならスキップ件数サマリーの定義更新

## 非対象

- 認証フロー
- タグ付与ロジックの仕様変更
- 勤務時刻 (`09:00-19:00`, `11:30-12:30`) の変更
