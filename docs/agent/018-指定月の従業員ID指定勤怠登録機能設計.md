# 018 指定月の従業員ID指定勤怠登録機能設計

## 要望

既存機能 "指定の月に自分の勤怠を詰め込む" に加えて、新機能 "指定の月に従業員ID指定で勤怠を詰め込む" を追加する。

- COMPANY ID は、アクセストークンに紐づくログイン従業員 (`users/me`) から取得する。
- EMPLOYEE ID は、コマンドライン引数で取得する。
- EMPLOYEE ID の入力時に、以下の案内を出す。
  - `よくわからんかったら Ctrl + C でいったん終わって、やり直してね`
- 勤怠 PUT の処理本体は、既存の "指定の月に自分の勤怠を詰め込む" と同等の流れで、指定従業員を対象に実行する。

## オーナーメモ

- 既存機能との差分は「対象 employee_id の解決方法」に寄せる。
- 既存ロジックを壊さない設計を優先する。
